# Weather App
This is a simple weather application that allows users to search for a city and view current weather information. The app automatically remembers the last searched city and reloads it on page refresh using localStorage.

## Features
- Search for weather by city name
- Display current weather data
- Automatically loads the last searched city
- API key handled internally (no user input required)
- Clean and responsive interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Weather API

## Project Structure
- `index.html` – Main HTML structure of the application
- `styles.css` – Styling and layout
- `script.js` – Application logic and API communication

## Setup Instructions
1. Download or clone the project.
2. Open the `index.html` file in your web browser.
3. The app will run immediately (API key is already included in `script.js`).

## Usage
1. Enter a city name into the search field.
2. View the weather information displayed on the screen.
3. Refresh the page – the app will automatically load the last searched city.

## Customization
- To change the API key, edit the constant in `script.js`.

## Notes
- Internet connection is required to fetch weather data.
- The app depends on the availability of the external weather API.
